# Asset Sources

## Best simple approach

Start with placeholders first so coding is never blocked.

- Player: green rectangle
- Cannibal: yellow rectangle
- Zombie: magenta rectangle
- Bullet: small white rectangle
- Background: blue-green solid color

After the game works, replace placeholders with real files.

## Where to get real assets

### Graphics

- Kenney: top-down and UI packs, easiest starting point
- OpenGameArt: extra characters, terrain, and effects
- Itch.io free assets: good if you want a different style

### Fonts

- Google Fonts: safest choice for a clean UI font
- Font Squirrel: good for title fonts

### Sound effects

- Freesound: gunshots, hits, footsteps, monster sounds
- Mixkit: quick free effects

### Music

- Incompetech
- FreePD
- OpenGameArt music section

## What one person should collect

One teammate can gather these:

- 1 player sprite
- 2 enemy sprites
- 1 background image or tile set
- 1 UI font
- 3 to 5 sound effects
- 1 day track
- 1 night track

## Folder plan

When assets are added, keep them here:

- `assets/textures`
- `assets/fonts`
- `assets/sounds`

## Rule for the team

No one should block on assets. If a file is missing, use a simple shape and keep building gameplay first.
