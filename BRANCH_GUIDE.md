# Premansh Branch Guide

## Your role

You own the **integration, assets, polish, and testing** side of the project.

## Read these files first

- `README.md`
- `TEAM_GUIDE.md`
- `CMakeLists.txt`
- `build.ps1`
- `run.ps1`
- `assets/`

## Your main responsibility

- make sure all parts of the project fit together
- organize assets, fonts, sounds, and placeholders
- improve polish and consistency
- test the full game flow and catch issues between systems

## Files mainly under your side

- `assets/*`
- build and run scripts
- final integration touches across the project

## What you should understand in the codebase

- how the project builds and runs
- how assets are loaded and where they belong
- how different systems depend on each other during gameplay
- where polish changes can be made without breaking ownership boundaries

## What others depend on from you

- consistent asset structure
- final integration between systems
- testing feedback and cleanup before submission

## Good reading order

1. `README.md`
2. `TEAM_GUIDE.md`
3. `CMakeLists.txt`
4. build and run scripts
5. `assets/` folder
